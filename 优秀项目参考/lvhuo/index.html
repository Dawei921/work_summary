草头饼

草头饼

Description

草头饼是一道美食，主要食材是新鲜草头、糯米粉。

  • 新鲜草头择洗干净，锅里烧开水，放一点点盐，草头放入沸水中焯一下
  • 将草头取出用冷水冲一下，挤干水分切碎放入碗中，撒上适量盐和味精，菜油烧烫之后浇入碗中拌匀
  • 糯米粉用温水和成团，盖上湿布稍微醒10分钟左右
  • 将糯米分成40g左右一个，包入草头馅料，拍扁，放入油锅煎至两面金黄即可

农家大扇贝

农家大扇贝

Description

扇贝解冻洗净，淋上料酒、李锦记蒸鱼豉油、麻油，码上蒜茸姜茸和红椒丁，静置片刻。用开水泡开粉丝，拉拉喜欢吃，就足足放了2捆，而且打定主意不放在上面，铺着。然后也给粉丝拌上料酒、蒸鱼豉油、蒜茸等，铺在大盘子上。摆上扇贝。蒸锅开了后，大火蒸5-6分钟左右。然后撒上细葱花。烧热2大勺油，哗啦啦哧哧淋在扇贝上，一股香气腾起，口水下来。真是好吃呀，特别是粉丝，量足过瘾。
